## Overview
Notes App is a full-stack web application with user authentication and CRUD operations for notes.
Users can register, login, manage their profile, and create/update/delete personal notes.
The backend is built with Node.js + Express and MongoDB, secured with JWT and bcrypt.
Advanced features include role-based access control (RBAC) and SMTP email integration.

## Tech Stack
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Security: JWT, bcrypt, Helmet, CORS
- Validation: Joi
- Email: Nodemailer (SMTP provider like SendGrid/Mailgun/Postmark)
- Frontend: Vanilla HTML/CSS/JS (Fetch API)
